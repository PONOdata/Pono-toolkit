A fork of Lenovo Legion Toolkit.

Pono Toolkit is a Windows utility for Lenovo Legion laptops. Most of it is upstream's work; what we changed is listed plainly on the changes page, and the people who built the foundation are on the upstream page. It is free, GPL-3.0, and not affiliated with Lenovo.

Install

  1. Download PonoToolkitSetup-v0.3.8.exe.
  2. Run it as administrator. It installs to C:\Program Files\PonoToolkit\.
  3. Launch from the Start menu and pick a language.

Pono Toolkit installs beside an existing Lenovo Legion Toolkit without touching it: its own directory, its own settings under %LOCALAPPDATA%\PonoToolkit\. Both can run on the same machine.
