Pono Toolkit is an AIO Windows desktop utility for power users. Lenovo Legion is the primary supported platform; the cross-vendor utilities (Dynamic Lighting, airplane toggle, automation) work on any Windows 11 device.

Fork of Lenovo Legion Toolkit. GPL-3.0. Independent.

What is in here that is not upstream

Auto-detect Dynamic Lighting

No debug flag. The Dynamic Lighting page surfaces automatically when a LampArray-conformant device is present.

System indicator effects

BatteryLow (red pulse below threshold), Charging (gradient sweep on AC), CapsLock indicator. Wall-clock sample throttle keeps cadence steady regardless of animation speed.

Borg auto-adapt

Zero-config adaptive effect for any LampArray device. Status and Branding lamps stay white; everything else runs a spatial rainbow whose period scales with lamp count. Resistance is futile.

Fn+F8 instant airplane mode

Flips the Windows airplane state directly via registry + Radio Management Service bounce. Notification confirms the new state. Falls back to opening the settings page if the registry write fails.

Smart key cycle sync

When an automation pipeline runs because of an event (game starts, etc.), the smart key cycle index advances past that quick action. The next press produces the next action instead of repeating.

Hot-path optimizations

Render loop runs at 30 Hz with cached capability probes, reused color/index buffers, and once-per-frame transition math. Lower CPU, smoother effects.

Latent crash fixes

Notification pipeline safe-defaults adding a new notification type is non-fatal. PowerModeExtreme arms restored across allow/symbol/text switches.

Three-tier signing pipeline

Build CI routes through PFX (canonical), Azure Trusted Signing (forks), or compile-check fallback (no secrets). Always meaningful CI signal.

Install

  1. Download the portable zip from the link above.
  2. Extract to a folder of your choice (e.g. %LOCALAPPDATA%\PonoToolkit).
  3. Run Pono Toolkit.exe as Administrator.

Pono Toolkit can coexist on the same machine as Lenovo Legion Toolkit. Different AppId, mutex, and settings folder (%localappdata%\PonoToolkit) keep installations independent.
