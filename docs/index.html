WHAT IS ADDED OVER UPSTREAM

Features that earn their place.

Pono Toolkit is a fork, not a rewrite. Most of the surface is upstream LLT. The differences below are the ones that move user-visible quality forward.

01

Auto-detect Dynamic Lighting

The page surfaces automatically when a LampArray-conformant device is present. No debug flag, no hidden setting.

02

Fn+F8 instant airplane mode

Flips the Windows airplane state directly via registry and a Radio Management Service bounce. Notification confirms. Falls back to opening Settings if the registry write is refused.

03

Smart-key cycle sync

When an automation pipeline fires because of an event (game launch, etc.), the smart-key cycle index advances past that quick action so the next press produces the next action, not a stale repeat.

04

System indicator effects

BatteryLow as a red pulse below threshold, Charging as a gradient sweep on AC, CapsLock as an indicator. Wall-clock sample throttle keeps cadence steady regardless of animation speed.

05

Hot-path optimizations

Render loop at 30 Hz with cached capability probes, reused color and index buffers, and once-per-frame transition math. Lower CPU, smoother effects.

06

Isolated AppData

Settings, logs, and integrations live under %LOCALAPPDATA%\PonoToolkit\. Existing Lenovo Legion Toolkit installs are untouched and can run side by side.

INSTALL

Run the installer.

  1. Download PonoToolkitSetup-v0.3.8.exe from the Download section above.
  2. Run as Administrator. Inno Setup completes silently in seconds and writes to C:\Program Files\PonoToolkit\.
  3. Launch from the Start menu. First run shows the upstream-shoutout dialog and a language picker; choose your locale and you are done.

Coexistence is intentional. Pono Toolkit installs to its own Program Files directory, uses a distinct AppId and mutex, and stores settings under %LOCALAPPDATA%\PonoToolkit\. Existing Lenovo Legion Toolkit installs continue to work on the same machine without interference. If you have prior Pono Toolkit settings under the legacy LLT-shared path, they migrate forward on first launch.
